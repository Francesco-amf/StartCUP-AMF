## Como Usar

1. Execute o script: `node create-evaluators.js`
2. Guarde estas credenciais em local seguro (Google Sheets, LastPass, etc)
3. Compartilhe com os avaliadores via email (não pelo chat)

## Notas Importantes

✅ **Sem acentos** em nenhuma senha
✅ **Case-sensitive** (maiúsculas e minúsculas importam)
✅ **@2024!** é obrigatório no final
✅ Emails usam domínio **@startcup-amf.com** (sem acentos também)

## Se Houver Problemas

Se alguém não conseguir fazer login:
1. Verifique se digitou corretamente (maiúsculas importam!)
2. Verifique se o navegador não está com cache antigo
3. Tente em modo incógnito
4. Verifique no Supabase se o usuário foi criado com provider=email
